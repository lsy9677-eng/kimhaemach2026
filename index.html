🔑 팀등록 비밀번호 관리
경기이사가 팀 등록 시 사용하는 비밀번호입니다. 간단한 번호로 설정 후 각 경기이사에게 알려주세요.
현재 팀등록 비번: ●●●●
• 관리자 비밀번호와 별도로 관리됩니다.
• 팀등록 비번으로는 팀 등록만 가능하며, 관리자 기능은 사용 불가합니다.
📱 관리자 연락처(문자 수신번호)
• 홈 화면의 “요청사항 문자 보내기” 버튼이 이 번호로 전송됩니다.
• 번호는 관리자만 변경 가능합니다.
📲 클럽 경기이사 연락처 관리
팀 등록 시 자동 저장됩니다. 직접 추가/수정도 가능합니다.
• 번호가 등록된 클럽에만 공지 문자 버튼이 활성화됩니다.
📨 경기이사에게 공지 문자 보내기
아래 버튼으로 내용을 자동 완성하거나 직접 입력 후 발송하세요.
🖼️ 대진표 이미지 저장 (카카오톡 공유용)
대진표 이미지는 문자 전송이 불가합니다. 이미지를 저장 후 카카오톡 단톡방에 직접 공유하세요.
🧾 2026 등록 선수 명단 관리
• 엑셀/텍스트 파일을 한 번에 올릴 수 있습니다.
• 클럽명은 자동으로 기본 클럽명으로 저장됩니다. 예: 모던A/모던B → 모던
• 2026년 이후 신규 대회는 등록 선수만 팀등록 가능합니다.

📥 과거 대회 엑셀 업로드

• 업로드한 엑셀에서 부서/클럽/선수명단/시합결과(1~3위)를 자동 인식해 저장합니다.
• 동명이인은 이름+클럽으로 분리 저장되어 혼동이 없습니다.
체크 해제 시: 기존 데이터 유지 + (부서+클럽)이 같은 팀은 자동 스킵하고 추가합니다.

➕ 선수 추가 (관리자)

• 동명이인/이적 혼동 방지를 위해 이름+클럽으로 저장합니다 (이름__클럽).
• 등록만 해두면 팀등록에서 바로 선택/확인할 수 있습니다.

📩 관리자에게 요청사항 보내기

• 휴대폰에서는 문자앱이 열립니다.
• PC에서는 메시지가 자동 복사되며, 휴대폰에서 붙여넣어 보내면 됩니다.
